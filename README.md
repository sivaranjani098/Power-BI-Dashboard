# Behavioral Analytics Dashboard – Screen Time vs Well-Being

This project presents an interactive Power BI dashboard that analyzes the relationship between screen time, productivity, sleep duration, and mental wellness across different demographic and device usage segments.

# Project Overview

The dashboard explores how digital usage behavior varies by age category, device type, usage purpose, gender, and living area, helping to understand its impact on overall lifestyle and well-being.

# Dataset

The dataset includes user-level behavioral data such as:

1. Age Category (Children, Teens, Early Teens)

2. Primary Device (Smartphone, TV, Laptop, Tablet)

3. Usage Type (Education, Entertainment, Balanced)

4. Screen Time, Learning Ratio

5. Productivity Score, Sleep Hours, Mental Wellness Score

6. Gender and Living Area

# Tools & Technologies

- Power BI – Data modeling & visualization

- DAX – Calculated measures and KPIs

- Data Cleaning & Transformation – Power Query

# Dashboard Highlights

- KPI Cards: Total Users, Total Screen Time, Avg Productivity, Avg Sleep Hours

- Behavioral Trends: Relationship between screen time and productivity

- Device Analysis: Screen time and learning ratio by device type

- Demographic Insights: Usage patterns by age category and gender

- Interactive Filters: Age group, living area, and usage type slicers

# Screen Time Behavior Overview

This page provides a high-level view of children's digital usage patterns. It highlights total users, overall screen time, average productivity score, and average sleep hours. Visuals show how screen time varies by device type, age category, and usage purpose (Education, Entertainment, Balanced). The goal is to identify general behavior trends and dominant usage patterns.

- Identifies dominant screen usage trends across different age groups and regions.
- Supports data-driven understanding of digital behavior distribution.
- Helps detect high screen-time segments for further analysis.

# Lifestyle & Well-Being Insights

This page focuses on how screen habits impact lifestyle indicators. It analyzes the relationship between screen time, productivity, mental wellness, and sleep duration. Comparative visuals help identify whether educational or entertainment usage leads to better well-being outcomes, enabling deeper behavioral analysis.
- Examines correlation between screen time and productivity levels.
- Analyzes the impact of digital habits on sleep duration.
- Helps identify balanced usage patterns linked to healthier outcomes.
- Enables behavioral insights for improving lifestyle management.

# Objective

To provide actionable insights into how screen usage habits influence productivity and well-being, supporting data-driven decision-making for lifestyle and behavioral analysis.
